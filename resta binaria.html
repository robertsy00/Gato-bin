Resta

Se realiza similar a la resta decimal, columna por columna, de derecha a izquierda. 0 - 0 = 0 1 - 0 = 1 1 - 1 = 0 0 - 1 = 1 (Bajo ciertas circunstancias se necesita "pedir prestado" de la siguiente columna) Por ejemplo: 1101 - 101 = 1000
